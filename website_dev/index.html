What is RenTales?


RenTales is an AI powered visual novel generator. RenTales takes a prompt from the user and converts it into a captivating visual story with character dialogues, character images, backgrounds, and music. The story generated will be played using Ren'Py to create an engaging experience. RenTales is made by using 4 different machine learning models to achieve text generation, image generation and music selection. RenTales is a unique demonstration on how various large language models can work together seamlessly to create the end result.

  • Narrative Generation: Input your prompts and RenTales will generate a story based on the prompt given.
  • Artwork Creation: RenTales uses context awareness models to generate character and background descriptions. RenTales then uses Stable Diffusion to create mesmerizing images for the characters and backgrounds. The image generation models are fine-tuned to create images in an anime style.
  • Soundtracks: Using context awareness machine learning models, RenTales assess the mood of the scene aand added a proper soundtrack from its library to match the scene.
  • Ren'Py Compatibility: RenTales uses Ren'Py story telling software to present its visual novel as the end product.
  • Deployment: Once your visual novel generation is complete, RenTales compiles everything into a Ren'Py script, ready for immediate deployment. Share your creation with the world at the click of a button.
